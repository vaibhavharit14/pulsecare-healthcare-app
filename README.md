# PulseCare VH - NGO Healthcare Support App

A concept-level web application designed to bridge the gap between healthcare volunteers and communities in need, serving as an assignment prototype for a healthcare NGO.

## 🚀 Tech Stack
- **Frontend Framework**: React + Vite
- **Styling**: Vanilla CSS with modern custom properties (CSS variables), Glassmorphism, and responsive design techniques.
- **Animations**: `framer-motion` for smooth, dynamic micro-interactions and transitions.
- **Icons**: `lucide-react` for crisp and consistent scalable vector graphics.

## 🤖 AI Automation Concept
**PulseAI Chatbot**
To reduce the manual workload of triaging patient or volunteer inquiries, PulseAI serves as a first-line automated respondent. 
- **Use-Case**: When users access the site, they are greeted by an AI assistant snippet. If they ask about **volunteering**, they are directed to the correct form path. If they require **medical support** or ask about **costs**, the AI provides immediate answers based on keyword recognition, routing them efficiently to the human support team if needed.
- **Form Triage**: The contact form simulates an AI classification system where the submitted requests are tagged by priority (e.g. Patient vs Partner NGO), preparing them for faster action by human administrators.

## ❤️ NGO Use-Case
- **Volunteer Registration**: Medical professionals can easily sign up and specify their availability through a streamlined process.
- **Patient Support**: Individuals seeking medical help can confidently submit their details knowing the automated triage will fast-track emergencies.
- **Resource Management**: Automating FAQ and initial contact reduces administrative bottlenecks, freeing up NGO resources for actual medical work rather than manual data entry.

## 🛠️ How to Run Locally

1. Clone or download the repository.
2. Ensure you have Node.js installed.
3. Open your terminal in the project directory.
4. Run `npm install` to install dependencies.
5. Run `npm run dev` to start the local development server.
